# Weather Forecast App

A responsive, real-time weather forecast web app delivering 5-day predictions for 100+ cities worldwide, with a clean, mobile-friendly UI.

## Overview

This app lets users search any city and view a real-time 5-day weather forecast, with dynamic icons that reflect current conditions and a one-click Celsius/Fahrenheit toggle.

## Tech Stack

- **HTML / CSS** — responsive, mobile-friendly layout
- **JavaScript** — API integration and dynamic UI updates
- **Weather API** — live weather data

## Features

- 5-day weather forecast for 100+ cities
- Dynamic weather icons based on current conditions
- Celsius / Fahrenheit toggle
- Reduced interaction steps by ~40% through simplified, mobile-first UI design

## Setup

```bash
git clone https://github.com/tshende2004/weather-forecast-app.git
cd weather-forecast-app
# open index.html in your browser, or serve locally:
npx serve .
```

You'll need a free API key from your weather data provider — add it to a `config.js` file (see `config.example.js`).
